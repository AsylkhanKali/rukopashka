# Сайт Всеказахстанской ассоциации рукопашного боя

Официальный одностраничный сайт национальной спортивной федерации (RU/KZ).
Чистый статический сайт: HTML + CSS + ванильный JavaScript, без сборки и зависимостей.

## Запуск

Просто откройте `index.html` в браузере. Для корректной работы шрифтов/иконок нужен интернет
(Google Fonts + Phosphor Icons по CDN).

Локальный сервер (рекомендуется):

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Структура

```
index.html              # вся разметка и контент (двуязычный: data-ru / data-kz)
assets/
  css/styles.css        # стили и дизайн-система (тёмная тема, золото + казахский голубой)
  js/main.js            # языки, меню, scroll-reveal, счётчики, список федераций
  img/                  # логотип, favicon и веб-оптимизированные фото
source/                 # исходники: Анкета_финал.docx и бланк с логотипом
images/                 # архив оригинальных фото с Instagram (не используется сайтом напрямую)
```

## Как менять контент

- **Тексты** — прямо в `index.html`. У переводимых элементов два атрибута:
  `data-ru="..."` и `data-kz="..."`. Меняйте оба.
- **15 региональных федераций** задаются массивом `REGIONS` в начале `assets/js/main.js`.
- **Язык по умолчанию — казахский**; выбор пользователя сохраняется в браузере (`localStorage`).
  Сменить дефолт можно в `assets/js/main.js` (функция `initLang`, значение `"kz"`/`"ru"`).

## Как заменить фотографии

Все изображения лежат в `assets/img/`. Чтобы заменить фото, положите новый файл с тем же именем
(желательно в том же соотношении сторон) — например `hero.jpg`, `about-training.jpg`,
`event-championship.jpg`. Использованные сейчас фото и их роли:

| Файл | Где используется |
|------|------------------|
| `hero.jpg` | Главный экран (фон) |
| `about-training.jpg`, `about-team.jpg`, `champion-podium.jpg` | Блок «Об ассоциации» (коллаж) |
| `event-championship.jpg`, `event-worldcup.jpg`, `event-trophies.jpg` | Блок «События» |
| `logo.png`, `favicon.png` | Логотип AKHFA и иконка вкладки |
| `sponsor-noc.png`, `sponsor-kazzinc.png` | Логотипы спонсоров (НОК РК, «Казцинк») |

Дополнительно подготовлены и доступны для замены: `about-fans.jpg`, `champion-flag.jpg`,
`champion-flowers.jpg`, `duo-flags.jpg`.

Фото оптимизированы для веба из материалов Instagram-аккаунта `@all_kazakhstan_h2h` и связанных
аккаунтов команд/спортсменов (папка `images/`).

## Что желательно уточнить перед публикацией

1. **Казахский перевод** текстов составлен и требует вычитки носителем языка.
2. **Фото руководства.** Карточки руководителей используют монограммы (инициалы). Автопоиск фото по
   ФИО в открытых источниках надёжного результата не дал (возвращаются групповые/событийные кадры и
   плейсхолдеры, а данные о должностях в вебе противоречивы), поэтому, чтобы не поставить лицо не того
   человека, оставлены монограммы. Пришлите официальные портреты — подставлю их вместо `.avatar`.
3. **Бухгалтер** в анкете указан без имени («Кенжегарина … Бирликовна») — добавьте полное имя.
4. **Даты соревнований** в блоке «События» указаны без точных чисел (только город/год) — уточните.
5. **Логотипы спонсоров:** НОК РК и «Казцинк» — реальные логотипы (с Wikimedia). HSIF показан
   текстовым знаком; при наличии официального логотипа HSIF можно заменить.
